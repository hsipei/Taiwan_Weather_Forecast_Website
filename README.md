# 台灣天氣預報網站

使用中央氣象署（CWA）開放資料平台 API 建立的天氣預報網站。

## 功能

- 📋 全台 22 縣市 36 小時天氣預報
- 🔍 依縣市篩選查詢
- 📱 響應式設計，支援手機與桌面瀏覽
- 🎨 美觀的天氣圖示與卡片式介面

## 技術架構

- **後端**：Node.js + Express
- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **資料來源**：中央氣象署開放資料平台 API

## 使用的 API 資料集

| 資料集代碼 | 說明 |
|-----------|------|
| F-C0032-001 | 一般天氣預報 - 今明 36 小時天氣預報 |
| O-A0003-001 | 自動氣象站觀測資料 |
